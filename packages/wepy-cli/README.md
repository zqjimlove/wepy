# wepy-cli-next

相比wepy-cli增加了以下功能：

* 多进程编译
* 编译缓存 ，关闭编译缓存 --no-cache
* -c,--clear 清空 dist 目录命令，此时不使用任何缓存

# 小程序框架wepy命令行工具

参见：[小程序框架wepy](https://github.com/wepyjs/wepy)


### 安装
```bash
npm install wepy-cli@next -g
```
